Page Numbers in Microsoft Word

Microsoft Word allows you to add headers and footers to your documents by double clicking in the top and bottom margins. These automatically appear on the top and bottom of every page in the document. You can make Word automatically add page numbers to each page in the document. This is especially helpful if you are writing anything with a table of contents.

When the header and footer are being edited, the rest of the document appears in a lighter color and this menu appears:


In these images, the header is shown. If you want to add the page number to the bottom of the page instead, scroll down and click inside the footer. Click 'Page Number' and then the 'Page Number' menu item to add a page number to the page.



Click the 'Format' button to find additional options regarding the page number (e.g. style, start point, etc.).
