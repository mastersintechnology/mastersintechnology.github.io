1. Creating a blank canvas.

[control/command + N] File → New → [select size + change file name] → OK → Save (As)


2. Crop

[C] Crop tool → [select area] → enter/return


3. Straighten

To straighten the image, select the level icon and draw the horizon line. Then, click ‘Enter/Return’ to apply your changes.

4. Improving lighting + Colo(u)r

Brightness/Contrast (half black/half white sun) → Auto/adjust at sliders
To view the image before your adjustments, click ‘Layers’ and then the eye icon to hide the layer.

5. Removing unwanted content

[J] Patch Tool:



Drag the cursor to the area you would like to base the patch off.

[J] Spot Healing Brush Tool






To switch between patch, spot healing, etc., click and hold the icon.

Common Keyboard Shortcuts

  • control/command + plus = zoom in
  • control/command + minus = zoom out
  • control/command + 0 = fit to screen
  • control/command + a = select all
  • control/command + t = select layer
  • control/command + b = auto-correct image
  • control/command + z = undo/redo
  • control/command + alt + z = step backward
  • left bracket = decrease brush size
  • right bracket = increase brush size
  • m = select tool
  • b = brush
  • e = eraser
