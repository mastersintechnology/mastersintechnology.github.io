Identifying Your Operating System

Back to all topics

A lot of what we do today is done online through web apps and services. However, different versions of the same operating system (OS) can have different features; standalone apps (the ones you download and use offline) can have different system requirements. Certain versions of applications may not support your operating system. Therefore, you should know how to identify the OS you are running.

To determine what version of Mac you are running, click on the Apple menu and click on "About this Mac".

A window will appear with the name and version number of your operating system. The image below is what would be shown on a Mid-2012 MacBook Pro running macOS Sierra (version 10.12) (the serial number has been covered for privacy reasons).

Mac operating systems likely to be in use today:

  • macOS Sierra - 10.12
  • Mac OS X El Capitan - 10.11
  • Mac OS X Yosemite - 10.10
  • Mac OS X Mavericks - 10.9
  • Mac OS X Mountain Lion - 10.8
  • Mac OS X Lion - 10.7
  • Mac OS X Snow Leopard - 10.6
Note: the change in branding of the operating system ("macOS" instead of "Mac OS X") is a decision made by Apple Inc. after the release of Sierra.
Lion, Mountain Lion, Yosemite, and El Capitan are most notable in terms of new features introduced. Some features described in other Mac 101 pages may not be available in all operating systems. Any features unavailable before a certain version of Mac will be indicated.
For a complete list of Mac operating systems, see the Wikipedia page "Macintosh Operating Systems".
